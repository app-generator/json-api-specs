# Json-API

Unified API specifications for all API Servers provided by AppSeed. API Methods:

- Auth API
    - login
    - logout
    - register
    - password-forgot
- Profile API
    - me [GET]
    - me [PATCH]
- Users API
    - users [GET]
    - users [POST]
    - users/{id} [GET]
    - users/{id} [PATCH]
    - users/{id} [DELETE]
- Items API
    - items [GET]
    - items [POST]
    - items/{id} [GET]
    - items/{id} [PATCH]
    - items/{id} [DELETE]
